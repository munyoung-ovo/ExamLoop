૮⑉･-･⑉ა **四个功能**

- 资料速通 — 上传PPT/PDF，自动提取考点和答题模板
- 出题练习 — 指定考点，按难度出题，自动批改
- 解题讲解 — 粘贴题目，分学科框架讲解
- 模拟出卷 — 生成完整试卷HTML文件
